# AccessOps brand

Logo and mark for AccessOps — accessibility conformance, operationalized.

## The concept
The mark fuses the two halves of the name. The **figure with outstretched arms inside a
circle** is the universal accessibility motif (Access); the **three nodes orbiting it**
read as an automated pipeline / continuous operations loop (Ops). The figure stays the
hero; the nodes imply the scan → draft → review → report cycle the product runs.

## Color
| Token | Value | Use |
|---|---|---|
| Indigo (accent) | `#4f56d3` | Mark tile, "Ops" in the wordmark, primary actions. |
| Ink | `#23252f` | "Access" in the wordmark, monochrome mark. |
| Muted | `#70717e` | Tagline, captions. |
| Reverse | `#ffffff` | Figure + nodes on the tile; wordmark on brand color. |

## Type
- Wordmark: **IBM Plex Sans** 600, letter-spacing −1. "Access" in ink, "Ops" in indigo.
- Tagline / micro: **IBM Plex Mono**.

## Clear space & minimum size
- Keep clear space of at least the mark's corner radius (≈16px at 64px) on all sides.
- Minimum mark size 24px; minimum lockup width 180px. Below that, use the mark alone.
- Don't recolor the figure, rotate the mark, add effects, or stretch the lockup.
